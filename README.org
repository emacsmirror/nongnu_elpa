#+TITLE: jabber.el - The XMPP client for Emacs
#+startup: overview


* About
=jabber.el= is an [[http://xmpp.org][XMPP]] client for Emacs.  XMPP is an IETF-standard
federated instant messaging protocol.

+ [[https://codeberg.org/emacs-jabber/emacs-jabber/][Project Page]]

+ chat: https://xmpp.link/#emacs@conference.conversations.im?join /general Emacs and jabber.el talk/
+ chat: https://xmpp.link/#jabber-el@conference.hmm.st?join /jabber.el development/

* Installation
#+begin_src emacs-lisp
  (use-package jabber
    :custom
    (jabber-account-list
     `(("example@server.im"
        (:password . ,(auth-source-pass-get 'secret "xmpp/server.im/example"))) ;;
       ("second@account.org"
        (:password . ,(auth-source-pass-get 'secret "xmpp/account.org/second")))))
    :config
    (jabber-modeline-mode 1))
#+end_src

+ Basic commands:
| Key           | Command                             |
|---------------+-------------------------------------|
| =C-x C-j C-c= | Connect all accounts                |
| =C-x C-j C-d= | Disconnect                          |
| =C-x C-j C-r= | Open roster buffer                  |
| =C-x C-j C-j= | Start or switch to a chat           |
| =C-x C-j C-m= | Join/switch to a MUC (groupchat)    |
| =C-x C-j C-b= | Switch to a chat buffer             |
